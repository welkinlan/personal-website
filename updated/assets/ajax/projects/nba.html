Projects Description

  • Our system allows 2 users to build their custom NBA teams by filling players on 5 positions (Center, PG, SG, PF, SF). Based on the players filled, the system calculates a score for each team and decides which team is better.
  • Web project built in Django (Python web framework), HTML/CSS/JS built with Bootstrap.
  • Ranking algorithm implemented in Python. Coefficients trained with linear SVM classifier on 1000 NBA players data (24 features) from official NBA statistics site
