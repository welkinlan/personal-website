Projects Description

  • Built an Android application assisting scientific researchers to capture certain diseases on murals by cameras in their mobile phones, add comments and send the images with the affiliated data to the image-collecting server.
  • Designed the overall interface of the application, combined different layouts together to make the interface more user-friendly.
  • Constructed the server of the project. Responsible for the transmission and storage of data. Encapsulated the data (JSON) collected in the phone using specific data structure. Stored the picture and the affiliated data into the database.
  • Joined the Google Android Application Challenge Contest on behalf of Tianjin University.
