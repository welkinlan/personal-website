EzSeminar

Projects Description

  • we introduce a convenient light-weight framework for users to manage seminars easily. This framework is adapted from the classical server-client framework, in which we introduce an Android mobile app as the light-weight client and develop the Apache web server as the database repository and data processing center.
  • We conducted a pilot user study on the students in TAMU CSE department, and the results showed that our system was easy and convenient to use.
